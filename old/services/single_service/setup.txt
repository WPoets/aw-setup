Curent Version: 1.0
Setup instructions
-------------------
1. Goto Awesome Core -> services module and add following lines, and save (most of these lines will be present, make sure last line is added if missing.)

[services.add single_service service_label='Single Service' post_type='single_service' desc='Single Service' /]

2. Now import single_service.xml via wordpress importer.

In case of updating

1. Trash the single_service modules and empty the trash
2. Import this xml file.
