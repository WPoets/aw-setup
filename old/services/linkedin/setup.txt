Curent Version: 1.1
Setup instructions
-------------------
1. Goto Awesome Core -> services module and add following lines, and save (most of these lines will be present, make sure last line is added if missing.)

[services.add linkedin_service service_label='Linkedin Service' post_type='linkedin_service' desc='Linkedin Service' /]
2. Now import linkedin_service.xml via wordpress importer.

In case of updating

1. Trash the linkedin_service  modules and empty the trash
2. Import this xml file.
